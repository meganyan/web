Why Make them?

Friendship bracelets are commonly given as gifts between teens and adults to symbolize eternal friendship and bond. They can vary from the loom, thread, color, and embroidery. Trying out colors and embroidery styles of these bracelets are not only fashionable but also beneficial. Making the bracelets would help sharpen your attention and meditate. It helps you pass times and focus on things at hand rather than other stressful factors in your life.

Calling all Beginners!

Most basic tools

  • strings of varying colors
  • tape and table or clipboard

For beginners choose strings of different colors, it reduces confusion! As you feel more comfortable with the pattern and the hand movements, you can choose strings of the same color. Might be helpful to have something that is sharp tipped (lead pencil or needles) to undo or untie the knots that are either ugly or a mistake.

The most basic or easy pattern to start with for beginners is a candy-striped or diagonal strip. It requires about 4 strings about 36 inches each. This style is to help you get comfortable with the basic flow of tying the most left string to the right into forwarding knots.

To make one forward knot

  • Using the most left string (string A)
  • Make a number 4 over the second most left string (string B)
  • Take the end of string A, put it underneath string B, and pull the end of string A up towards the right.
  • Make number 4 again using string A over string B and pull it up towards the right

Getting Fancy and Cool!

The more complicated pattern than the candy strip is called the chevron bracelet. This style requires about 4 strings of 72 inches each. Fold the strings in half and fasten a knot at the closed end. There should be a loop after you tie your knot. This style creates a letter V shape by tying the outermost left string into the middle using the forward knots and tying the outermost right string into the middle using the backward knots.

Backward knots is going in the opposite direction of the forward knot
To make one backward knot

  • Take the most right string (string A)
  • Make a number four over the second most left string (string B)
  • Take the end of string A, put it underneath string B, and pull it up towards the left.
  • Make number 4 again using string A over string B and pull it up towards the left.

There is the Chinese ladder bracelet that needs 3 strings. Unlike the chevron and candy-striped, the Chinese ladder uses 1 string and ties the rest of the 2 strings using the forward knot.

There is the double basic braid that needs 3 colors, 2 each, in a total of 6 strings. This one is simple and the end results look like braided hair.

There is the zig-zag bracelet that requires 4 strings of colors. The colors can be of your choice but for a cool 3D look go to this site. That site also teaches you how to tie the knots for the cool patter as well!
